# Transportation Problem Solver

## Возможности

- **Гибкий ввод данных** с настраиваемыми размерами матрицы
- **Два алгоритма решения**:
  - **Метод северо-западного угла** (Northwest Corner Method)
  - **Метод минимальных элементов** (Minimum Cost Method)
- **Визуализация результатов** в табличном формате
- **Пошаговое описание решения** с подробными вычислениями
- **Автоматическая проверка баланса** задачи
- **Пример данных** для быстрого тестирования
- **Интуитивно понятный пользовательский интерфейс**

## Технологии

- **Язык:** C#
- **Платформа:** .NET Framework / .NET Core
- **UI:** WPF (Windows Presentation Foundation)
- **Архитектура:** MVVM-подход

## Установка и запуск

### Требования:
- .NET Framework 4.7.2 или выше / .NET Core 3.1+
- Visual Studio 2019+ или другое IDE с поддержкой C#

### Запуск:
1. Клонируйте репозиторий:
```
git clone https://github.com/your-username/transport-problem-solver.git
```
2.Откройте решение в Visual Studio:

```cd transport-problem-solver
TransportProblemApp.sln
```

3. Соберите и запустите проект (F5)

## Математическая основа

### Транспортная задача:
Минимизировать целевую функцию:

```Z = ΣΣ cij × xij → min```
При ограничениях:

```
Σ xij = ai    (запасы)
Σ xij = bj    (потребности)
xij ≥ 0
```
### Методы решения:
- **СЗУ:** Последовательное заполнение от северо-западного угла.
- **Мин. элементы:** Приоритет клеткам с минимальной стоимостью.
